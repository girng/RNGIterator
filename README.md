# RNGIterator

Calculates the amount of iterations needed for a chance to be true in [Crystal](https://crystal-lang.org/)


## Usage
 - crystal RNGIterator.cr --release
 - You can modify the 2 parameters (chance, times) to your liking. Enjoy!
